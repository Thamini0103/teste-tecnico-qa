# Teste Técnico QA

## Objetivo
Automação de testes funcionais utilizando Cypress para validação dos principais fluxos da aplicação ServeRest.

## Fluxos automatizados
- Login
- Cadastro de usuário
- Cadastro de produto
- Exclusão de produtos

## Tecnologias utilizadas
- Cypress
- Javascript
- Github Actions

## Execução do projeto

Instalar dependências:
npm install

Executar Cypress:
npx cypress open

Executar em modo headless:
npx cypress run

## Evidências

As evidências de execução podem ser visualizadas diretamente na execução do Cypress e na pipeline configurada no Github Actions.
